# ECCO API Examples

This repository contains example applications demonstrating how to integrate with the ECCO API for AI-powered image generation.

## Examples

### 1. Flask Web Application (`flask_webapp/`)
A simple Flask web application that demonstrates image generation using the ECCO API.

**Features:**
- Web-based interface for image generation
- Support for Nano Banana and Banana Pro models
- Real-time image generation and display

**Setup:**
```bash
cd flask_webapp
pip install -r requirements.txt
python app.py
```

See [flask_webapp/README.md](flask_webapp/README.md) for detailed instructions.

### 2. Flutter Mobile App (`flutter_app/`)
A cross-platform Flutter application for mobile image generation.

**Features:**
- Native mobile experience (iOS & Android)
- Model selection (Nano Banana & Banana Pro)
- Image generation with custom prompts
- Clean, modern UI

**Setup:**
```bash
cd flutter_app
flutter pub get
flutter run
```

See [flutter_app/README.md](flutter_app/README.md) for detailed instructions.

## Getting Started

1. Clone this repository:
```bash
git clone https://github.com/5haa/eccoapi-examples.git
cd eccoapi-examples
```

2. Choose an example and follow its specific README for setup instructions.

3. Make sure you have your ECCO API key ready for configuration.

## Requirements

- **Flask Example**: Python 3.7+
- **Flutter Example**: Flutter SDK 3.0+
